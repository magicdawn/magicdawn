# 颜色空间 color space

- sRGB, 包括 (`rgb()` / `hsl()` / `hsv()`)
- Display P3, 包括 (`p3()` / `display-p3()`), 属于 RGB 空间
- CIE Lab, 包括 (`lab()` / `lch()`)
- OKLab, 包括 (`oklab()` / `oklch()`)
